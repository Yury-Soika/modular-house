# Modular Prefab Houses Landing Page

## Project Overview

Create a modern, high-converting landing page for a company specializing in modular timber-frame houses.

The website should focus on:

- generating qualified leads;
- showcasing completed projects;
- presenting the company's manufacturing capabilities;
- explaining the construction process;
- building trust;
- encouraging users to request a consultation or receive a catalog of house plans.
